Pertanyaan jujur

Dan jawaban jujur.

Kenapa gw gak bayar agency aja Rp 200 juta?

Karena lu bayar mereka untuk overhead 50 orang, project manager, sales tim, kantor di SCBD. Gw solo founder — gw kerjain langsung. Trade-off-nya: gw gak bisa handle 100 klien sekaligus, scope gw lebih kecil dan specific. Kalau project lu butuh enterprise SLA + dedicated team, gw bukan pilihan tepat.

Kenapa gw gak build sendiri pakai ChatGPT API?

Lu bisa. Tapi production-grade AI customer support butuh: prompt engineering yang gak halu, RAG retrieval akurat, eval harness untuk validate output, fallback ke human, idempotency untuk action (refund dll), policy gate. Itu 3–6 minggu trial-error solo. Kalau lu engineer dan ada waktu — gas, gw rekomendasiin. Kalau enggak, gw udah lalui semua jalur itu, lu beli waktu.

Gimana kalau hasilnya jelek?

Review Analyzer: kalau output generic atau gak actionable, refund 100%, no questions asked. WA Bot Setup: kalau gw gak deliver dalam 14 hari kerja, refund 100%. Strategy Workshop: kalau setelah sesi lu rasa gak dapat insight baru, refund 50%. Gw lebih rugi waktu kalau salah deliver — incentive gw align.

Apa beda gw sama Cekat, Wati, Qontak, Mekari?

Mereka SaaS subscription model — bayar tiap bulan, mereka host, gak ada source code. Bagus kalau lu mau plug-and-play. Gw productized service model — bayar sekali, lu pegang source code, gak ada vendor lock-in, gak ada per-conversation fee. Trade-off: lu yang tanggung hosting + maintenance (atau bayar gw retainer kecil).

Lu siapa? Apa bukti track record?

Gw solo founder, baru launch. Track record yang gw punya: working demo di hero atas — coba sendiri sekarang, beneran jalan. Code di GitHub, infrastructure di Supabase + Vercel. Bukti lain belum ada — gw baru mulai. Makanya gw tawarin paket Rp 750rb dulu sebagai entry point. Lu test, kalau bagus lanjut.

Bayar gimana? Aman?

Bayar transfer bank (BCA / BNI / Mandiri) atau e-wallet. Setelah order form di-submit, lu dapet email dengan detail rekening + invoice. Konfirmasi via WhatsApp setelah transfer. Gw mulai kerjain max 1 hari kerja setelah konfirmasi. Untuk paket besar (custom), invoice formal + termin 50/50.

Apa data customer gw aman?

Untuk Review Analyzer: lu kasih gw URL publik, gw scrape review publik — gak ada data sensitif. Untuk WA Bot Setup: lu pegang infrastructure sendiri, KB lu gak masuk database gw. Gw gak akses customer data lu kecuali lu explicitly share untuk debug. DPA / NDA bisa di-sign untuk project >Rp 15jt.
